# mir-lapack
NDSLICE wrapper for LAPACK

## Required static libraries
 - `lapack` - FORTRAN 77 LAPACK API

## Wrapped API

 - gelsd
 - gesdd
 - gesv
 - gesvd
 - getrf
 - getri
 - spev
 - sptrf
 - sysv_rook
 - sptri
 - potri
 - pptri
 - trtri
 - tptri
